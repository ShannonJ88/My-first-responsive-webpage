My Books

Posted July 17, 2017

Listed below, are pictures and descriptions of some of my favorite books. These books can be booth informative and fun to read. If you think you would enjoy reading any of these books and you would like to purchase one, just click on the picture and it will take you to Amazon.

The C Programming Language By Dennis M Ritchie and Brian W Kerrighan is an excellent book for both new computers and expert programmers. The book is packed with many examples, and is yet small and concise. This book is written by the person that created the language, so there is no better source of information I highly recommend this book to anyone who wishes to become a C programmer.

HTML and CSS is a great book for new comers to Web Design. This book is packed with lots of examples, and is easy to read for beginners and challenging enough for experienced programmers and web designers. It is a must read and Elizabeth Castro made the format a step-by-step guide with crystal-clear instructions to introduce you to all of the HTML and CSS essentials. Click on the image for more details about how you can purchase this book.

Database Systems is my favorite book to learning SQL and the principles of Database Design. SQL is the primary language used for talking to a database, and this book explains the language in detail. Whether you are a novice or advanced SQL developer, you will learn a lot form this book. This is the ideal book for Database Systems and Database Design and Application courses. Click on the image for more details about how you can purchase this book

Java Programming could not be easier with this book. This was my first book I ever read when it came to learning Java. I can tell you from experience that I have NEVER read a better book than this one when it came to learning Java. Java is a tough language to learn but this book makes it easy. This book is a comprehensive coverage of Java and programming which makes it a useful reference for beginning programmers and IT professionals.
